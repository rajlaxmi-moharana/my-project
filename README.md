# Restaurant Menu Project

A responsive restaurant menu website built using HTML and CSS.

## Live Demo

 https://rajlaxmi-moharana.github.io/my-project/

## Features

- Responsive design
- Mobile-friendly layout
- Portrait mode support
- Landscape mode support
- Smooth navigation
- Menu sections for food, desserts, and beverages

## Recent Updates

- Added portrait media queries
- Added landscape media queries
- Improved mobile responsiveness

## Technologies Used

- HTML5
- CSS3
